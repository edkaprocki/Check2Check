Welcome to Check2Check by QuickSoft, Inc.

Thank you for trying Check2Check. This unique budget and checking program allows you to be in control of your finances by letting you plan your future spending. There are many excellent checking programs on the market but none will show you as quickly and easily as Check2Check where your money is going. The ability to try "what-if" scenarios is key to any program that is used for budget planning and Check2Check does this superbly. You'll be able to enter your income and expenses months ahead of time and even put in future expenses such as car insurance, vacations, etc. and Check2Check will show you if you'll be able to meet your future financial obligations.


1. Unregistered / Registered Version
Check2Check is released as an unregistered version which means that it will operate with full features for 30 days at which time you must register it if you continue to use it. Certain functions are disabled after the 30 day trial period but if the program is registered they will automatically be re-enabled. To register Check2Check first bring up the program and then click the "Help | Buy Now" menu item and you can then click on the "Buy Now" button which will take you to our online order web page. Once registered you will be given a registration code to enter in the appropriate box on the registration dialog form.


2. Installation
Installation is simple with only a single installation file. You can run this file by double clicking on it and following the instructions. All files that are extracted during the installation process will be deleted to ensure your computer doesn't get cluttered up. 


3. Uninstallation
To uninstall Check2Check bring up the Control Panel and double click Add/Remove Program. Select Check2Check and click Add/Remove and all files associated with this application will be removed.
